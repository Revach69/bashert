# TDD Workflow Skill

## Cycle: RED -> GREEN -> REFACTOR

1. Write failing test
2. Write minimal code to pass
3. Refactor while keeping tests green

## Framework: Vitest
```bash
npm test
npm run test:watch
npm run test:coverage
```

## Coverage: 80% minimum, 100% for critical paths
